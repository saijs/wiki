无效字符

Language Message
简体中文 无效字符
繁体中文 不正確的字元
字元無效
英文 Invalid character
日文 文字が正しくありません。
俄文 Недопустимый знак

使用被 JavaScript 编译器识别为无效的一个或多个字符来组合标识符。 有效字符使用下列规则：

  • 第一个字符必须是 ASCII 字母（大写或小写）或下划线 (_)。
  • 随后的字符可以是 ASCII 字母、数字或下划线。
  • 标识符名称不能是保留字。

应该避免使用不属于 JavaScript 语言定义的字符。
